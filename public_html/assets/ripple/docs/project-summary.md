# RIPPLE Project Summary (from `PROPOSAL.pdf`)

## Project At A Glance
- Project: RIPPLE - Reducing Impact of Plastic Pollution Through Learning and Engagement
- Call: HORIZON-MISS-2025-03-OCEAN-02
- Type: Horizon Europe proposal (toolbox for public authorities, river-to-ocean plastics/litter)
- Duration: 48 months (M1-M48)
- Core mission: Reduce plastic pollution through a combined model of citizen participation, open tools, pilot validation, digital monitoring, and replication.
- Main pilot geographies: Spain (Besos), Greece (Remathia), Croatia/Bosnia and Herzegovina basin context (Neretva).

## What WPs Are
Work Packages (WPs) are the main execution blocks of the project plan.  
Each WP has a lead partner, timeline, effort (person-months), objectives, tasks, deliverables, and milestones.

## Four-Phase Execution Logic (WP-Focused)
- Phase 1 (Foundation, M1-M12): WP1, WP2, WP3, WP4 (architecture start), WP5 (protocol/data pipeline setup).
- Phase 2 (Prototyping and Preparation, M6-M18): WP2 prototyping/co-design, WP3 early training modules, WP5 dashboards, WP6 pilot preparation.
- Phase 3 (Demonstration and Validation, M12-M30): WP6 full pilot deployment, WP5 monitoring/impact data, WP4 consolidation into practical toolbox content, WP3 iteration.
- Phase 4 (Replication and Uptake, M18-M36 and continuation to M48): WP7 Associated Regions (ARs), policy/exploitation/replication pathways, WP4+WP5 final integrated toolbox/platform; WP8 keeps governance/compliance.

## Practical Reading Of The Workplan
- WP1-WP3 build social and capacity foundations.
- WP2-WP5 build and operationalize technical/digital solutions.
- WP6 validates in real local conditions.
- WP4 packages validated methods/tools into an authority-ready toolbox.
- WP7 drives scale-up, policy uptake, and exploitation.
- WP8 ensures project control, quality, ethics, and compliance.

## Quick Takeaways
- The project is intentionally WP-integrated: no WP is standalone.
- Highest execution weight is in WP6 (pilots), then WP4/WP5 (toolbox + digital platform).
- By design, the final value is a validated and replicable toolbox, not only pilot activities.
